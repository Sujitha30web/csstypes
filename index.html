The w3-container class is the most important of the W3.CSS classes. It provides equality like:

    Common margins
    Common paddings
    Common vertical alignments
    Common horizontal alignments
    Common fonts
    Common colors
    The w3-container class is typically used with HTML container elements, like:
    
    div, header,footer,article,section,blockquote, form,and more.

INLINE STYLE SHEET

An inline style may be used to apply a unique style for a single element. To use inline styles, add the style attribute to the relevant element. The style attribute can contain any CSS property.



INTERNAL STYLE SHEET

An internal style sheet may be used if one single HTML page has a unique style. The internal style is defined inside the element, inside the head section.>



EXTERNAL STYLE SHEET

An external style sheet can be written in any text editor, and must be saved with a .css extension. The external .css file should not contain any HTML tags.
